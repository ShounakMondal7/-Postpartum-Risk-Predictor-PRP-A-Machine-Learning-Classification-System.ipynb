# -Postpartum-Risk-Predictor-PRP-A-Machine-Learning-Classification-System.ipynb
ML pipeline that predicts postpartum depression risk from self-reported symptoms. Compares Logistic Regression, Random Forest &amp; SVC on a composite-scored dataset of 11 maternal mental health indicators — built to catch risk hidden behind evasive or ambiguous survey answers. 98% recall on the held-out test set.
